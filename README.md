# Thrustcurve

ThrustCurve.org model rocket motor and thrust curve data as a single JSON file

## Installation

You know the drill ...

```
npm i thrustcurve
```

## Usage

```
import thrustcurve from 'thrustcurve';

// `thrustcurve` is a Motor[] array. See `thrustcurve.d.ts` for the structure of Motor objects
```
